## 🎮 Di cosa si tratta

**MakeItWash** è un gioco gestionale sviluppato da [Endless-Team](https://github.com/Endless-Team).
Controlla robot e nastri trasportatori per lavare, smistare e gestire oggetti
in un impianto industriale che diventa sempre più caotico.

## 🚀 Come avviarlo

### Requisiti
- Java 17+
- Gradle (o usa il wrapper incluso)

### Avvio
```bash
git clone https://github.com/Endless-Team/MakeItWash.git
cd MakeItWash
./gradlew desktop:run
```

Su Windows:
```bash
gradlew.bat desktop:run
```

## 🏗️ Struttura del progetto

```
core/       → logica di gioco (entità, schermate, mondo)
desktop/    → launcher per PC
```
